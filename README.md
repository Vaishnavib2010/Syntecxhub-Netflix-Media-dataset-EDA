# 📊 Netflix / Media Dataset EDA

This project performs **Exploratory Data Analysis (EDA)** on a movie dataset to understand trends in movie releases, genre popularity, and audience ratings. The analysis is done using Python and data visualization libraries.

---

## 📌 Project Objectives

- Explore movie **release trends by year**
- Identify the **most common genres**
- Analyze **popularity and vote average distribution**
- Generate **Top-10 lists** for genres and years
- Visualize insights using charts

---

## 🛠️ Technologies Used

- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 📂 Dataset Features

The dataset contains the following columns:

- Release_Date  
- Title  
- Overview  
- Popularity  
- Vote_Count  
- Vote_Average  
- Original_Language  
- Genre  
- Poster_Url  

---

## 📊 Visualizations Included

- Movies released per year (trend analysis)
- Top 10 movie genres
- Popularity distribution
- Vote average distribution
- Top 10 years with most movie releases

---

## 🔍 Key Insights

- Movie releases increased significantly in recent years.
- **Drama, Action, and Comedy** are among the most common genres.
- Most movies have a **vote average between 6 and 7**.
- Only a small number of movies reach **very high popularity levels**.
